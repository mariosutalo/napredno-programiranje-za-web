procedura za one koji nemaju zadnju verziju projekta
- skinuti moj repo napredno-programiranje-za-web-3 na radnu površinu te raspakirati
- u svom projektu uraditi commit i push ako imate ikakvih promjena
- iskopirati mapu 6. web shop i zalijepiti je u svoj repozitorij (ako imate mapu 6. web shop potrebno ju je prije toga izbrisati)
- u .gitignore dodati ako nije dodano redak:6. web shop/node_modules
- u terminalu se prebaciti u mapu 6. web shop
- u terminalu pokrenuti naredbu npm i
- pokrenuti projekt:node --inspect --watch-path=./ ./index.js
- nodemon verzija : nodemon index
